# Livox Mid-360 blueprints

- mid360-fastlio-voxels-native — SLAM + C++ global voxel map at 3Hz
    
# Using USB to Ethernet Adapter
If you are using an adapter make sure to use the following commands so that the host
can succesfully communicate with the Mid-360:
- run `ip -br link` to see if the adpater is recognized. You should see an ip similar to enx00e04c680125
- then assign a static IP to the adapter using `sudo ip addr add 192.168.1.5/24 dev <adapter IP>`

Make sure the adpater is fully connected with the ethernet cord. Now you should be able to use any 
Mid-360 blueprint.
